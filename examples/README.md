# Examples

This directory will contain example Open Specifications that demonstrate the
standard in action.

## Planned Examples

- **URL Shortener** — A simple service showing the basics of spec writing
- **Task Queue** — A backend system showing infrastructure-level specification
- **AI Experiment Factory** — A complex system showing Level 5 autonomous development
